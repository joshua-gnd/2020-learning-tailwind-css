Custom Styling

go to style and under components type custom components: .customstyle { @apply bg-blue-600 text-white; /* conventional CSS can be added */ border: 2px solid black; }
Download stylelint extension
In terminal type "npm install stylelint -config-recommended --save-dev"
Go to settings and remove check on "CSS: Validate"
type "npm run build:css" in terminal (something like GULP can be used to listen for changes instead of going to the terminal)

Thu 30 Apr, 2020 vue

installing vue: in terminal; npm install -g @vue/cli
check for version: vue --version
terminal: vue create vjs_class (vjs_class is the name)
terminal: cd vjs_class
terminal: npm run serve
Output:
App running at:
- Local: http://localhost:8080/
- Network: http://192.168.0.11:8080/
Note that the development build is not optimized.
To create a production build, run npm run build.
